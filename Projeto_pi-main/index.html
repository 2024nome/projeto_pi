Projeto Integrador - Jogo 2D: O Livro Perdido

Este projeto visa o desenvolvimento de um jogo 2D ambientado na biblioteca do SENAC, onde o personagem principal encontra um livro antigo e é transportado para um universo paralelo.
O jogo terá classes jogáveis como Bárbaro, Bardo, Mago e Druida, cada uma com habilidades específicas para resolver puzzles. O objetivo é criar uma experiência imersiva, combinando narrativa envolvente e mecânicas de jogo desafiadoras.
Este trabalho faz parte do Projeto Integrador (PI), com foco na aplicação de conceitos de programação, design e desenvolvimento de jogos 2D.
